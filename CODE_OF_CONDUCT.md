# Code of Conduct

We aim for a respectful, professional environment.

**Be respectful.** Assume good intent. No harassment, personal attacks, or discrimination.
**Be constructive.** Focus on technical facts and reproducible steps.
**Protect users.** Do not post sensitive data or PII in issues/PRs.
**Report security issues privately** as described in `SECURITY.md`.

Violations may result in comments being limited or contributors being blocked.
